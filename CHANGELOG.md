# Changelog – Aura Beauty OS

All notable changes to this project will be documented in this file. This project adheres to Semantic Versioning (`vMAJOR.MINOR.PATCH`).

---

## [v2.0.0] – 2026-06-20 (Aura OS Beauty Operating System Overhaul)

### Added
- **3D Cyber-Face Assistant**: Added WebGL wireframe model canvas utilizing raw Three.js with idle breathing, thinking, and speaking animations.
- **AI Face Scan Simulation**: Camera stream canvas with glowing laser overlays mapping mock diagnostics metrics back to profile models.
- **Leaflet Smart Maps**: Added OpenStreetMap dark-theme canvas showing local salon waitlist queues and driving distance ETAs.
- **AI Budget Combo Optimizer**: Knapsack service package compiler matching budget constraints.
- **User & Admin Authentication**: Added registration modal with preference filters and admin privilege logic.
- **Refer & Earn Dashboard**: Referral credit card links sharing code ledgers.
- **AI Beauty Profile subtab**: Added customized dashboard segment displaying styling choices and diagnostic records.

### Changed
- **Matte Dark Luxury Theme**: Replaced standard color schemes with premium carbon-matte backgrounds (`#040406`) and champagne gold typography (`#D5C4A1`).
- **Geometric Vector Monogram Logo**: Replaced placeholder styling with a premium vector logo in header blocks.
- **Reorganized Header Navigation**: Dynamically structures menu options based on active user credentials.
